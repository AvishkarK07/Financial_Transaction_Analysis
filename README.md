- **Total Records:** 5,000

- **Time Period:** Transactions recorded across multiple dates in 2024  

---

## ğŸ§¾ Column Description

| Column Name | Data Type | Description |
|------------|----------|-------------|
| `Transaction_ID` | Text | Unique identifier for each transaction |
| `Customer_ID` | Text | Unique identifier for each customer |
| `Transaction_Date` | Date | Date when the transaction occurred |
| `Amount` | Numeric | Transaction amount |
| `Transaction_Type` | Text | Type of transaction (Credit, Debit, Transfer) |
| `Payment_Method` | Text | Payment mode (Card, UPI, etc.) |
| `Merchant_Category` | Text | Category of merchant (Retail, Travel, Healthcare, Entertainment, etc.) |
| `City` | Text | City where the transaction took place |
| `Account_Balance` | Numeric | Account balance after the transaction |
| `Fraud_Flag` | Numeric (0/1) | Fraud indicator (0 = Normal, 1 = Fraudulent) |
